- **公共 / 社会インフラ**
  - 公共交通をサポートするツール（運行情報・経路最適化・観光案内・高齢者向けナビなど）
- **ビジネス / 業務 AI**
  - 中小企業で今すぐ使えるツール（小さな業務改善の即効性ある AI ツール群）
  - 中小企業の AI 導入をサポートするツール
  - 製造業をサポートするツール
  - マーケティング自動化ツール
  - 会社運営をするエージェント集（Skills として CEO / CFO / 人事 / 経理 などの役割別に整備）
  - 業界の業務を置き換える AI ツール（士業・医療・建築・物流など、業界特化型）
- **家事 / 暮らし**
  - 家計簿レシピツール（家計と食材／レシピを連動させる）
  - 子どもの学校プリント情報管理ツール
  - 料理レシピの生成・管理ツール（給食メニューや付け合わせとの組み合わせ提案）
  - 1週間の献立ツール
  - 1週間のコーディネート（服装）ツール
- **教育 / 学習**
  - 子どもの金融リテラシーを育てるツール
  - 教育ツール（英語、算数などの科目別学習）
- **ヘルスケア / コーチング**
  - モチベーションをサポートする AI
  - 運動サポートツール
  - ゴルフや野球などのスポーツアドバイザー（フォーム解析・練習メニュー提案など）
- **AI 時代の◯◯（既存サービスの再発明）**
  - AI 時代のニュースサイト（自分にパーソナライズされたニュースを Podcast / Text / 動画で配信）
  - AI 時代のイベントサイト（テック系。connpass / Luma 的なもの）
  - AI 時代のメーラー（Gmail / Outlook 的なメールクライアントの再発明）
  - AI 時代のカレンダー（Google Calendar の再発明）
  - AI 時代の家計簿アプリ（マネーフォワード / Zaim 的）
  - AI 時代のタスク管理 / TODO アプリ
  - AI 時代の地図／ナビアプリ
  - AI 時代の RSS リーダー／情報収集ツール
  - AI 時代のグルメサイト（食べログ的なものの再発明）
- **パーソナル / ライフハック**
  - アプリ・パソコン・音声のメモを1ヶ所に集約し、簡単に検索できるツール
  - 位置情報に基づくメモ／通知ツール（例: 「薬局に行ったら〇〇を買う」とメモしておくと、近くになったら教えてくれる）
  - Picasa のような便利な、ローカルで動く写真ツール
- **金融 / 個人ファイナンス**
  - 投資系のツール
  - 確定申告をより簡単にするツール
- **AI エージェント / LLM 応用**
  - LLM の context management ツール
- **GraphAI**
  - GraphAI の agent を追加する
  - GraphAI 本体の改良
  - GraphAI の graph／フローを簡単に作成するための補助ツール
- **MulmoCast / マルチメディア**
  - MulmoCast を使った動画サービス／Podcast サービス／プレゼンテーションツール、その他 wow なツール
  - MCP + AI + ffmpeg などを組み合わせて、無人もしくは簡単に動画編集できるツール／アプリ。長尺素材から切り出して YouTube 向けやショート動画化するもの
- **開発者ツール / プラットフォーム**
  - Vibe coding 時代にふさわしいターミナル（動作中の Claude Code の状態や、どこで動かしているかなど、そのとき必要な情報が見えるターミナル）
    - 作業中の dir がわかる
    - 直近で何をしているかわかる（プロンプトで送った内容が見えると良い）
  - Raycast 拡張
  - BaaS（Firebase, Supabase など）を使って、非エンジニアがセキュアなプロジェクトを作れるようにサポートするツール（skill でも汎用ツールでも可）
