## 作り方の考え方

- **DOM（ドム）** … ブラウザが持つ「画面の部品ツリー」。JSはこれを書き換えて表示を変える。→[02](02-vanilla.md)
- **命令的（めいれいてき）** … 「こう操作しろ」と手順を1つずつ書くやり方（素のJS）。→[01](01-overview.md)
- **宣言的（せんげんてき）** … 「状態がこうなら画面はこう」と書き、変化は道具が反映するやり方（React/Vue）。→[01](01-overview.md)
- **リアクティブ** … 状態が変わると**自動でUIが追従**する性質。→[03](03-react.md)
- **コンポーネント** … UIを再利用できる**部品**に分ける単位。→[03](03-react.md)
- **仮想DOM（かそうドム）**（🔧） … 変更を一度メモリ上で計算し、本物のDOMには**最小限だけ**反映して速くするしくみ（React/Vue）。Svelte/Solidは使わない。
- **JSX（ジェイエスエックス）** … JSの中にHTMLっぽく書ける記法（React）。→[03](03-react.md)
- **SFC（単一ファイルコンポーネント）** … テンプレ・スクリプト・スタイルを1つの `.vue` にまとめる書き方（Vue）。→[04](04-vue.md)

## 描画戦略（どこで画面を作るか）

- **SPA（シングルページアプリ）** … ブラウザのJSが画面を組み立てるアプリ。→[06](06-spa.md)
- **CSR（クライアントサイドレンダリング）** … SPA の描画方式。ブラウザ側で描く。→[06](06-spa.md)
- **MPA（マルチページアプリ）** … 昔ながら。ページごとにサーバーがHTMLを返す。→[01](01-overview.md)
- **SSR（サーバーサイドレンダリング）** … サーバーでHTMLを作って返す。初回が速くSEOに強い。→[07](07-ssr.md)
- **SSG（静的サイト生成）** … ビルド時にHTMLを作り置きしてCDNから配る。最速・最安。→[08](08-ssg.md)
- **ISR（増分静的再生成）**（🔧） … 古くなったページだけ後から作り直すSSGの発展。→[08](08-ssg.md)
- **ハイドレーション** … サーバーが作った静的HTMLに、ブラウザでJSをかぶせて“動く”ようにすること（乾物に水）。→[07](07-ssr.md)
- **RSC（React Server Components）** … Reactの部品の**一部をサーバーで実行**するしくみ。秘密やDBをサーバー側に置ける。→[09](09-metaframeworks.md)
- **Server Actions（サーバーアクション）** … フォーム送信などを**サーバーの関数**で直接処理するしくみ。→[09](09-metaframeworks.md)
- **アイランド（islands）**（🔧） … 基本は静的HTMLで、動く所だけ部分的にJSを効かせる（Astro）。→[09](09-metaframeworks.md)
- **ハイブリッド** … ページごとにCSR/SSR/SSGを混ぜること。→[09](09-metaframeworks.md)
- **メタフレームワーク** … React/Vueの上で描画戦略やルーティングを全部まとめる土台（Next.js/Nuxt/Astro）。→[09](09-metaframeworks.md)
- **ストリーミング / エッジレンダリング**（🔧） … 部分的に早く返す／CDNの“端っこ”で描画して速くする工夫。

## 道具・周辺

- **バンドラ** … たくさんのファイルを少数にまとめる道具。→[10](10-build-host.md)
- **トランスパイル** … TS→JS、新しい構文→古い形へ“翻訳”すること。→[10](10-build-host.md)
- **Vite（ヴィート）** … いま定番のビルドツール。開発が爆速。→[10](10-build-host.md)
- **ホスティング** … 作ったものを置いて配信する場所。→[10](10-build-host.md)
- **CDN（しーでぃーえぬ）** … 世界中に置いた配信用の倉庫。近くから届くので速い。
- **SEO（エスイーオー）** … 検索で見つけてもらいやすくすること。SSR/SSGが有利。
- **TTFB**（🔧） … 最初の1バイトが返るまでの時間。サーバー処理の速さの目安。

## 名前（フレームワーク・ツール）

- **React（リアクト）** … Meta製。コンポーネント＋JSX、求人最多。→[03](03-react.md)
- **Vue（ビュー）** … 学びやすい宣言的フレームワーク。→[04](04-vue.md)
- **Svelte（スベルト）** … コンパイラ型、記述が少ない。→[05](05-other-frameworks.md)
- **Solid（ソリッド）** … React風＋超高速。→[05](05-other-frameworks.md)
- **Angular（アンギュラー）** … Google製の全部入り、企業向け。→[05](05-other-frameworks.md)
- **jQuery（ジェイクエリー）** … いにしえの立役者。今もWordPress等で現役。→[05](05-other-frameworks.md)
- **Next.js（ネクスト）** … Reactのメタフレームワーク。SSR/SSG/RSC。→[09](09-metaframeworks.md)
- **Nuxt（ナクスト）** … Vueのメタフレームワーク。→[09](09-metaframeworks.md)
- **Astro（アストロ）** … 静的＋アイランドで軽い。→[09](09-metaframeworks.md)
