- **未来テクノロジー / 次世代サービス**
  - 自動運転時代のアプリケーション（自動運転が前提となった世界の上で動くサービス群）
  - ゲーミフィケーションを応用して人間の行動を良い方向に誘導するサービス（ナッジ／行動デザイン）
- **メディア / SNS / 言論プラットフォーム**
  - エコーチェンバー・炎上のない、まっとうな SNS
  - 多様な意見を可視化できるソーシャルメディア
  - 右派・左派・海外からの視点をきちんと入れられるメディア
  - 物事を歴史的背景・科学的知見にもとづいて解説してくれる動画サービス
  - 嘘の意見・非科学的な話を判定してくれるツール（ファクトチェック AI）
- **Web3 / 分散型サービス**
  - 国家に依存しない、安定性・信頼性があってボラティリティの少ない平等な crypto
  - 実用的な Web3 サービス
- **セキュリティ / 安全実行環境**
  - ISO/IEC 27001（ISMS）や P マークなどのセキュリティ認証取得・運用を AI で安価にサポートするサービス
  - Firebase / GCP / AWS / Azure などのクラウド環境をセキュアに使うのを補助するツール（IAM 設計・公開設定チェック・コスト爆発防止など）
  - フリーランス向けセキュリティツール（会社ごとにセキュアな環境を1台のコンピュータで構築する）
  - Claude Code などを安全に動かすツール（chroot/サンドボックスで隔離実行）
  - npm などのパッケージをローカルで安全に動かすためのツール（依存ライブラリの実行隔離）
- **公共 / 社会インフラ**
  - 公共交通をサポートするツール（運行情報・経路最適化・観光案内・高齢者向けナビなど）
- **ビジネス / 業務 AI**
  - 中小企業で今すぐ使えるツール（小さな業務改善の即効性ある AI ツール群）
  - 中小企業の AI 導入をサポートするツール
  - 製造業をサポートするツール
  - マーケティング自動化ツール
  - 会社運営をするエージェント集（Skills として CEO / CFO / 人事 / 経理 などの役割別に整備）
  - 業界の業務を置き換える AI ツール（士業・医療・建築・物流など、業界特化型）
- **家事 / 暮らし**
  - 家計簿レシピツール（家計と食材／レシピを連動させる）
  - 子どもの学校プリント情報管理ツール
  - 料理レシピの生成・管理ツール（給食メニューや付け合わせとの組み合わせ提案）
  - 1週間の献立ツール
  - 1週間のコーディネート（服装）ツール
- **教育 / 学習**
  - ゲームをしているうちに気付いたら賢くなる学習ゲーム
  - 悪影響のない子ども向け動画を生成するエンジン（安全フィルタ付きの教育動画生成）
  - AI でパーソナライズされていく演習問題
  - AI で教科書のつまずきを検出してサポートするツール
  - テキストから動画を生成して、わかりやすく説明するツール
  - プリント／テストから類題を作成するツール
  - 過去問から推測して予想問題を作るツール
  - ユーザーが間違った理由を推測し、どこを間違って覚えているかを補助するツール
  - 物理・英語・数学などの関連分野で弱い部分を推定し、そこを強化するツール
  - 子どもの金融リテラシーを育てるツール
  - 教育ツール（英語、算数などの科目別学習）
- **ヘルスケア / コーチング**
  - モチベーションをサポートする AI
  - 運動サポートツール
  - ゴルフや野球などのスポーツアドバイザー（フォーム解析・練習メニュー提案など）
- **AI 時代の◯◯（既存サービスの再発明）**
  - AI 時代のニュースサイト（自分にパーソナライズされたニュースを Podcast / Text / 動画で配信）
  - AI 時代のイベントサイト（テック系。connpass / Luma 的なもの）
  - AI 時代のメーラー（Gmail / Outlook 的なメールクライアントの再発明）
  - AI 時代のカレンダー（Google Calendar の再発明）
  - AI 時代の家計簿アプリ（マネーフォワード / Zaim 的）
  - AI 時代のタスク管理 / TODO アプリ
  - AI 時代の地図／ナビアプリ
  - AI 時代の RSS リーダー／情報収集ツール
  - AI 時代のグルメサイト（食べログ的なものの再発明）
- **パーソナル / ライフハック**
  - アプリ・パソコン・音声のメモを1ヶ所に集約し、簡単に検索できるツール
  - 位置情報に基づくメモ／通知ツール（例: 「薬局に行ったら〇〇を買う」とメモしておくと、近くになったら教えてくれる）
  - Picasa のような便利な、ローカルで動く写真ツール
- **金融 / 個人ファイナンス**
  - 投資系のツール
  - 確定申告をより簡単にするツール
- **AI エージェント / LLM 応用**
  - LLM の context management ツール
- **GraphAI**
  - GraphAI の agent を追加する
  - GraphAI 本体の改良
  - GraphAI の graph／フローを簡単に作成するための補助ツール
- **MulmoCast / マルチメディア**
  - MulmoCast を使った動画サービス／Podcast サービス／プレゼンテーションツール、その他 wow なツール
  - MCP + AI + ffmpeg などを組み合わせて、無人もしくは簡単に動画編集できるツール／アプリ。長尺素材から切り出して YouTube 向けやショート動画化するもの
- **開発者ツール / プラットフォーム**
  - Vibe coding 時代にふさわしいターミナル（動作中の Claude Code の状態や、どこで動かしているかなど、そのとき必要な情報が見えるターミナル）
    - 作業中の dir がわかる
    - 直近で何をしているかわかる（プロンプトで送った内容が見えると良い）
  - Raycast 拡張
  - BaaS（Firebase, Supabase など）を使って、非エンジニアがセキュアなプロジェクトを作れるようにサポートするツール（skill でも汎用ツールでも可）
